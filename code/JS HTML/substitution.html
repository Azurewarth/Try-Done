虽然大部分媒体鲜有提及，但根据《体育画报》报道，勇士顾问纳什也在这次招募杜兰特的战役中送出了自己的助攻。之前勇士方面对拉来杜兰特有一定信心，他们主要觉得杜兰特虽然在个人方面忠诚于维斯布鲁克，但在球场上，他其实已经很厌倦维斯那种长时间霸球的打法，而库里展示的是一种自由流动的进攻，这是两位顶级控卫之间最大的不同。但多位勇士教练、高层都认为，如果不是纳什的助攻，杜兰特来不来还很难说。“没有纳什的话，我不确定我们是否能得到杜兰特。”其中一位人士表示。在这次6队招募杜兰特的会晤中，勇士呈现的东西并没有特别高于其它对手。勇士总经理迈尔斯也表示：“我们听说快船深深打动了他，而我们却没有做到。”
